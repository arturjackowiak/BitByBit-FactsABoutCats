# FactsABoutCats
- fetch 10 facts about cats from https://catfact.ninja
- add/delete/modify facts
- get random facts

## Stack
* Frontend:
  * TypeScript
  * React
  * React Hook Form
  * Styled Components
  * React Router Dom
  * rsuite
  * yup
  * vite
  
## Installation and Setup Instructions

Clone down this repository. You will need `node` and `yarn` installed globally on your machine.  

Installation:

`yarn`  

To Start Server:

`yarn dev`
